# 📖 Project Overview

This project predicts employee salaries using **Simple Linear Regression**.

The primary objective was not only to build a prediction model but also to understand the complete Machine Learning workflow including:

- Data Exploration
- Data Cleaning
- Feature Selection
- Model Training
- Model Evaluation
- Performance Interpretation

This project marks the beginning of my Machine Learning journey.

---

# 🎯 Business Problem

Companies often want to estimate employee salaries based on years of experience.

Instead of manually estimating salaries, a Machine Learning model can learn the relationship between experience and salary and make predictions for new employees.

---

# 📊 Dataset Information

- Rows : 29
- Columns : 2
- Target Variable : Salary
- Feature : YearsExperience

There were no missing values in the dataset.
